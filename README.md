# Ads Click Coin - Telegram Mini App

এটি একটি টেলিগ্রাম মিনি অ্যাপ যা ব্যবহারকারীদের বিভিন্ন টাস্ক পূরণ করে কয়েন উপার্জন করার সুযোগ দেয়। এই প্রজেক্টটি HTML, CSS, এবং JavaScript ব্যবহার করে তৈরি করা হয়েছে এবং সুপাবেস (Supabase) ডাটাবেসের সাথে সংযুক্ত।

## প্রজেক্টের বৈশিষ্ট্য:
- **ড্যাশবোর্ড:** ইউজারের মোট ব্যালেন্স দেখার ব্যবস্থা।
- **টাস্ক:** নতুন টাস্ক সম্পন্ন করার অপশন।
- **রেফারেল:** বন্ধুদের আমন্ত্রণ জানিয়ে বোনাস পাওয়ার সিস্টেম।
- **ওয়ালেট:** ব্যালেন্স উইথড্র করার সুবিধা।
- **প্রোফাইল:** ইউজারের ব্যক্তিগত তথ্য।

## টেকনিক্যাল স্ট্যাক:
- **Frontend:** HTML5, CSS3, JavaScript (Telegram WebApp SDK)
- **Database:** Supabase
- **Deployment:** Vercel

## কীভাবে ব্যবহার করবেন:
১. এই রিপোজিটরিটিকে আপনার গিটহাবে ক্লোন করুন।
২. ভার্সেল (Vercel) থেকে এই রিপোজিটরিটিকে কানেক্ট করুন।
৩. টেলিগ্রাম বটের সাথে আপনার অ্যাপের লাইভ লিঙ্কটি লিঙ্ক করুন।
